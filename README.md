Focus Mode is a VS-Code Extension, which helps you stay concentrated and productive will youre programming.
Made like the Pomodoro Learning Method. 

## Features

1. Timer to track your time while working
2. Start/Stop/End it at the bottom of the window.
3. Change the timer duration in the Settings (Open Command Palette and run Preferences: Open Settings (JSON).)
4. Block certain file formats to help you stay in your actual window. (For example: when I'm learning/understanding the code i wrote and the solution is in a.md file it doesn't let you open it so you don't peek xD)
5.Dashboard to show your streak and Milestone rewards (3, 5, 10, 20)


## How This Helps You As A Developer

Stay focused while working
Use a well-known learning technic
